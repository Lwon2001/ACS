===== Lecture note =====

=== 1. Mitchell's definition of Machine Learning ===
	Machine learning occurs when performance **P **of algorithm at task **T** improves with experience **E**.
	
	* Learning task: regression, classification, transcription, sampling, etc.
	* Performance measure: accuracy, fitness, optimum, etc.
	* Experience: supervised learning, unsupervised learning, reinforcement learning

1. Categorise a given list of machine learning problem as //supervised, unsupervised or reinforcement learning.//
